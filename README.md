# Meal Mate 🍽️

Meal Mate is a full-stack food ordering and restaurant management web application that enables users to browse restaurants, explore menus, add items to a cart, and place food orders seamlessly.

## Features

* User Registration and Authentication
* Restaurant and Menu Browsing
* Food Search Functionality
* Shopping Cart Management
* Order Placement and Tracking
* Admin Dashboard for Restaurant Management
* Responsive User Interface
* Secure Data Handling

## Installation

### Clone the Repository

```bash
git clone https://github.com/pavanjii/meal-mate.git
cd meal-mate
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Database Migrations

```bash
python manage.py migrate
```

### Start the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```text
http://127.0.0.1:8000/
```
